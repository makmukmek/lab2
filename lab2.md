# Калькулятор Геометрических Фигур

Консольное приложение для расчёта геометрических параметров трёхмерных фигур с сохранением результатов в базу данных и различные форматы отчётов.

## Возможности

- **Расчёт параметров фигур**:
  - Параллелепипед (объём, площадь поверхности, масса)
  - Тетраэдр (объём, площадь поверхности, масса)
  - Шар (объём, площадь поверхности, масса)

- **Поддержка материалов**:
  - Сталь (7850 кг/м³)
  - Алюминий (2700 кг/м³)
  - Медь (8960 кг/м³)

- **Сохранение результатов**:
  - SQLite база данных
  - Текстовые файлы (.txt)
  - CSV файлы (.csv)
  - История расчётов

- **Дополнительные функции**:
  - Статистика расчётов
  - Просмотр истории
  - Docker-контейнеризация

## Требования

- Python 3.9+
- Или Docker

## Установка

### Способ 1: Локальная установка

1. **Установите зависимости**:
```bash
pip install -r requirements.txt
```

2. **Запустите приложение**:
```bash
python main.py
```

### Способ 2: Запуск через Docker

1. **Сборка образа**:
```bash
docker build -t geometry-calculator .
```

2. **Запуск контейнера**:
```bash
docker run -it --rm geometry-calculator
```


## Использование

### Главное меню

```
ГЛАВНОЕ МЕНЮ
--------------------
1. Новый расчёт
2. Посмотреть последние результаты
3. История расчетов
4. Статистика
5. Сохранить последний отчёт
6. Выход
```

### Процесс расчёта

1. **Выберите фигуру**:
   - Параллелепипед (требует длину, ширину, высоту)
   - Тетраэдр (требует длину ребра)
   - Шар (требует радиус)

2. **Выберите материал**:
   - Сталь, Алюминий или Медь

3. **Получите результаты**:
   - Объём (м³)
   - Площадь поверхности (м²)
   - Масса (кг)

4. **Сохраните результаты** (опционально):
   - В базу данных
   - В текстовый файл
   - В CSV файл
